# @web-remarq/mcp

MCP server for [web-remarq](https://github.com/DPostnik/web-remarq) — gives AI
agents (Claude Code, Cursor, Codex, Windsurf) direct access to project
annotations via the cloud-mode backend.

## Status

`0.1.0` — initial release. Cloud-mode only. 5 tools.

Documentation will be expanded in Task 12 of the implementation plan.
